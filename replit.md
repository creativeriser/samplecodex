# Project Overview
This is a React + Vite project with a dashboard/visit management focus.

## Project Structure
- `src/components/ui/`: Shadcn UI components.
- `src/pages/`: Main application pages (Dashboard, Login, Browse, etc.).
- `src/data/`: Mock data or data services.
- `src/styles/`: Global CSS.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Lucide React (icons)
- Shadcn UI

## User Preferences
- Clean, modular component structure.
- TypeScript for type safety.
